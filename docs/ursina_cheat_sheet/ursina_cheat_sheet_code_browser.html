--------------------------------
 getting_started
--------------------------------
    
    This document is good for looking up attribute names and classes
    you just can't remember the names of. Just press Ctrl+F and search away!
    The 'Entity' class is a useful place to start and is kind of like
    a GameObject(Unity) or an Actor(Unreal).
    Also, don't forget to check out 'global_variables' for useful functions
    like raycast(), invoke(), duplicate(), and destroy().
    
    
    
    example:
        from ursina import *
        app = Ursina()
    
        b = Button(text='hello world!', color=color.azure)
        b.scale *= .2
        b.tooltip = Tooltip('click me')
    
        app.run()

--------------------------------
about_scripts
--------------------------------
    scripts can be added to Entities like this:
        entity.add_script(MyClass())
     or entity.add_script(MyClass) # makes an instance of the class
     or entity.add_script("script_name") # makes an instance of first class in script
    
    scripts gets assigned a variable called 'entity'.
    target entity gets assigned a variable with the same name as the script.
    
    example:
    camera.add_script(SmoothFollow(offset=(0,10,-10)))
    print(camera.smooth_follow)
